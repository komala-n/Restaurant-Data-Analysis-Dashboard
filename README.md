# 🍽️ Restaurant Data Analysis Dashboard

## 📊 Project Overview
This project is an interactive Power BI dashboard created to analyze restaurant data including sales, customer behavior, and performance metrics.

## 🔍 Key Insights
- Top-performing restaurants based on revenue
- Most popular food categories
- Customer visit trends over time
- Peak hours for orders
- City-wise restaurant performance

## 📈 Features
- Dynamic filters (City, Category, Time)
- KPI cards for total sales, orders, and ratings
- Visual analysis using charts and graphs

## 🛠 Tools Used
- Power BI
- Data Cleaning
- Data Visualization
